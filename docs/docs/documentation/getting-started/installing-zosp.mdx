# Installing the Zelthy Open Source Platform (ZOSP)
To get started with the ZOSP,  you will need to install it using the Python package manager, pip. The installation process is simple and can be done with a single command.

Please ensure you have Python and pip installed on your system. If you haven't installed them yet, you can find Python installation instructions [here](https://www.python.org/downloads/) and pip installation instructions [here](https://pip.pypa.io/en/stable/installation/).

Once Python and pip are installed, you can install the Zelthy Open Source Platform by running the following command in your terminal or command prompt

```shell
pip install zosp
```

This command will download and install the latest version of ZOSP along with its dependencies.
